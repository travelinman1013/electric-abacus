Key Formula

Access Control Examples

✓ ALLOWED: User A accesses /businesses/biz-111/ingredients/ground-beef
JWT: { businessId: "biz-111" } Rule Check: "biz-111" == "biz-111" → ✓ ALLOW Result: Returns ground-beef data (Alice's Restaurant only)
✗ DENIED: User A attempts /businesses/biz-222/ingredients/coffee-beans
JWT: { businessId: "biz-111" } Rule Check: "biz-111" == "biz-222" → ✗ DENY Result: Permission Error - User A CANNOT access Bob's data

Automatic Query Scoping

User B's React App

  • useBusiness() → { businessId: "biz-222" }
